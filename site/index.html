DW:DO

DW:DO - the webapp (new version)

The webapp is a webpage, installable as an app on your phone, which helps you to pray the Ordinariate's Mattins, Evensong and Compline according to the Commonwealth Edition of Divine Worship: Daily Office.

How to use it:

Open the office page.

  • You will immediately see today's office - if it's the morning, Mattins will be shown; if it's after noon you'll see Evensong. Compline appears after 9pm - this is configurable (see below).
  • If there are options for the day - usually optional memorials - these will be listed below the day and date. You can tap/click on any option to display the office for that memorial.
  • If it's a Sunday, the introduction to Mattins or Evensong is shown as well.
  • If you've left it open on your device since last time you'll need to refresh it
  • In the toolbar, you will see a letter indicatiung the office (M for mattins, etc). If you click/tap that letter, it will cycle through Mattins/Evensong/Compline
  • You can select from options as you go down the page, where appropriate: for example, most days you can choose between the Venite or the Jubilate as the invitatory at Mattins - just tap on the appropriate tab.

The calendar

  • In the toolbar there is a calendar icon (exactly what it looks like depends on your device).
  • Tap on this to be taken to the calendar for the current liturgical year
  • When you see the calendar it is set close to today - so if you want to look earlier in the year scroll up.
  • You can tap on any day to show the office page for that day, or use the buttons next to the year to move forward/backward a year
  • If just browsing the calendar, you can use the 'return' button in the toolbar to return to today's office.
  • If you've selected another date, a 'reload' icon will appear on the toolbar - this will return you to today.

Other Texts

The 'document' icon in the toolbar will take you to a page which allows you to pray the Litany, or to select a Psalm to see its full text.

Settings

In the toolbar there is a settings icon - a gear shape. Tap on this to be taken to the settings

Here you can set:

  • Whether the officiant is usually a priest. This will pre-select the 'lay' options where these differ
  • Your country. This will affect the calendar. (In the US you will still get British English spellings!)
    • If in Canada or Australia, you can select whether to use the Commonwealth prayer instead of the Royal ones
  • The size of text to suit your needs
  • Dark Mode: if you prefer a dark look, perhaps especially in the evening, you can set it here, for each Office.
  • At what time to start to show Compline instead of Evensong
  • Whether to show the Marian Anthems after Mattins and/or Evensong. (They always appear for Compline).
  • Whether you prefer the anthems in Latin or English. Like the officiant option, this pre-selects the appropriate version; you can still switch during the office if you wish.
  • Page buttons. If you use an e-reader and are used to using buttons to 'turn pages', this emulates that behaviour.

How to install:

(Installing allows offline use, but you can always use it online without installing if you prefer)

  • If you've been using the previous version, uninstall it
  • In your web browser, open the office page
  • After visiting a couple of times, the browser will give you the option to install the webapp
  • If that doesn't appear, check the browser's menu and look for an install option.
  • Operating systems and even versions of operating systems differ on exactly what options they show!
